principais formatações

Nesta frase temos um negrito usando B.
Nesta outra frase temos um texto em destaque usando STRONG.
Nesta frase temos um itálico usando I.
Já nesta um termo em ênfase usando EM.
Podemos criar um texto marcado usando a tag MARK.
Podemos criar um texto grande usando a tag BIG (mesmo estando obsoleto), e um texto piqueno usando a tag SMALL.
Podemos criar um texto excluído (usando DEL) e um texto inserido (usanso INS).
Existe também o sublinhado usando a tag U.
Para inserir coisas no subrescrito, como: x20+3, usamos a tag SUP;
para inserir coisas no subscrito, como: H2O, usamos a tag SUB.
